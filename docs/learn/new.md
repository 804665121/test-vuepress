# Composition API的优势

1. #### Options API 存在的问题(vue2) 配置式

   使用传统的Options API 新增或者修改一个需求,就需要分别在data,methods,computed里修改

   ```
   export default{
      data(){},
      methods:{},
      computed:{}
   }
   ```
